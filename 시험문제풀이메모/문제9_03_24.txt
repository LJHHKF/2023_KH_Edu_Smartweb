[문제1]
화면의 색상이 적혀있는 버튼을 클릭하면 그 색상에 해당하는 박스가 나타나도록 작성한 코드이다. 박스가 나타나지 않는 원인과 조치내용을 작성하세요.

<문제점>
1. 'document.getElementsByTagName("button")'으로 리턴되는 값은 배열인데, 이 변수를 아래에서 단일 항목 다루듯이 다루고 있습니다. 배열 방식으로 고쳐야 합니다.

이 외 다른 문제점은 찾지 못하였습니다.

<해결책>
1. 반복문 for을 이용하여 'btn[i].addEventListener("click", fucntion(){~});' 방식으로 수정하였으며, 이를 적용 후 정상 동작하는 것을 확인했습니다.

[해결 후 코드]
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <script>
        window.onload = function () {
            var btn = document.getElementsByTagName("button");
            for (let i = 0; i < btn.length; i++) {
                btn[i].addEventListener("click", function () {
                    color = document.getElementById("color");
                    color.style.width = "100px";
                    color.style.height = "100px";
                    color.style.background = this.innerText;
                });
            }
        }
    </script>
</head>

<body>
    <p>
        색상을 선택하세요
        <button>Red</button>
        <button>Orange</button>
        <button>Yellow</button>
        <button>Green</button>
        <button>Blue</button>
    </p>
    <div id="color"></div>
</body>

</html>


[문제2]
아래의 요구사항을 반영하는 화면을 구현하고자 했지만, 완성된 화면과 다른 결과를 얻었따. 각 요구사항별 코드가 잘못된 원인을 작성하고 조치내용을 작성하시오.
(단, 문제해결은 JQuery를 이용하시오)

1. 이 문서를 처음 로드하면 로그인 화면은 보이지 않는다
2. label 태그는 가로로 '80px' 크기를 갖는다
3. 로그인 버튼을 클릭하면 로그인 화면이 화면에 나타난다
4. Close 버튼을 클릭하면 로그인 화면이 사라진다.

<문제점>
1. 스크립트 초반부에 $('label').css(~)를 적용하는 부분에서, {}를 하여 객체로써 css를 만들어 전달해줘야 하는데 그냥 바로 입력하고 있습니다. 수정해줘야 합니다.
또한 이 문제는 명세서 '2. label 태그는 가로로 '80px' 크기를 갖는다' 가 적용 안된 이유이기도 합니다. 해결하기만 하면 같이 해결됩니다.

2. 이벤트를 등록하려는 것 같은데, 핸들러를 함수가 아닌 JQuery로 넣어뒀습니다. 에러 오류도 핸들러 관련해서 나오고 있는 듯 하여, 조정해줘야 합니다.

3.각 click 이벤트 관련 JQuery 셀렉터를 (button)이나 (btn cancel) 등으로 작성해두었습니다. 이러면 'button' 태그인 모든 객체에 적용되는 등의 문제가 발생하므로, 수정해줘야 합니다.
또한 앞의 2번과 이 3번 문제를 해결하면 명세서의 '3~4번', 버튼 기능 관련 문제들이 해결됩니다.

4. '1. 이 문서를 처음 로드하면 로그인 화면은 보이지 않는다'가 적용이 안 되어 있습니다. 적용해야 합니다.

<해결책>
1. 해당 부분의 앞과 뒤에 {}를 추가하여 해결하였습니다.
또한 이를 통해 '2. label tag는 가로로 80px 크기를 가진다'도 해결되었습니다.

2. 익숙한 .on(click, (핸들러 함수)) 방식으로 해보려다가 이 문법 자체는 처음봐도 틀린 건 아닌 것 같아서 핸들러 동작부를 'fucntion(){~}'으로 감싸봤습니다(='.click(function(){~});'. 정상 동작합니다.

3-1 첫번째 ('button')은 동작상 ('.open')버튼 클래스를 얘기하는 것 같아서 이 방식으로 수정했습니다.
3-2.두번째 '(btn cancel)'은 ('.btn .cancel')을 얘기하는 것 같으나 다중 클래스 셀렉터가 기억나지 않아 ('.cancel')으로만 수정하였습니다. 현재로썬 문제 없이 동작합니다.

또한 이 3번 및 2번을 통해 명세서의 '3~4번', 기능 문제들이 해결되었습니다.


4. 이미 앞서 만들어둔 이벤트를 활용해서 아래에 '$('.cancel').click()' 부분을 추가하여 시작하면 기본적으로 열려있는(display가 block인) 상태인 것을 닫힌(diplay가 none인) 상태로 바꾸도록 하였습니다.

[해결 후 코드]
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <script src="http://code.jquery.com/jquery-3.3.1.min.js"></script>
    <style>
        form .btn{
            background-color: #4CAF50;
            color: white;
            padding:10px 20px;
            border:none;
            cursor: pointer;
            width:100px;
            margin-bottom:10px;
            margin-top:10px;
        }
        button.btn:hover{
            opacity: 0.8;
            color: yellow;
        }
        .cancel:hover{
            opacity: 0.8;
            background:red;
        }
    </style>
</head>
<body>
    <h2>로그인을 클릭하면 로그인 폼이 나타납니다.</h2>
    <button class="open">로그인</button>
    <div class="form-popup" id="myForm">
        <form action="result.html" method="post">
            <fieldset>
                <legend>로그인</legend>
                <h1>Login</h1>
                <label for="email"><b>Email</b></label>
                <input type="text" placeholder="Enter Emaile" name="email" required>
                <br>
                <label for="psw"><b>Password</b></label>
                <input type="password" placeholder="Enter Password" name="psw" required>
                <br>
                <button type="submit" class="btn">Login</button>
                <button type="button" class="btn cancel">Close</button>
            </fieldset>
        </form>
    </div>

    <script>
        $('label').css({
            'display' : 'inline-block',
            'width' : '80px'
        });
        $('.open').click(function(){$('#myForm').css('display','block');});
        $(".cancel").click(function(){$("#myForm").css('display','none');});
        $('.cancel').click();
    </script>
</body>
</html>
